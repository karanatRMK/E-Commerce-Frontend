# E-Commerce Website

A Ecommerce Website made with React.js Framework.


## Features

- Easy to integrate with Backend
- Fully Responsive


## Run Locally

Clone the project

```bash
  git clone https://github.com/karanatRMK/E-Commerce-Frontend
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
